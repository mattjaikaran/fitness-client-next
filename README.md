# Fitness Client

## Technologies
- NextJS
- React Bootstrap
- SCSS
- Next Auth
- Axios

## Getting Started

First, run the development server:

```bash
git clone https://github.com/mattjaikaran/fitness-client-next
cd fitness-client-next
yarn
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
